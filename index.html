IoT Dev

Datalogger & Monitoring

A compact and efficient datalogging system for real-time and historical energy monitoring. It communicates with smart energy meters over Modbus RS-485 using byte or raw hex requests, logging responses to embedded ROM or an external SD card. The system supports hybrid storage and syncs data to the cloud via HTTP APIs. It features a user-friendly dashboard for both online and offline monitoring. Built using ESP-32 with MicroPython and Beaglebone running Python both headlessly.
